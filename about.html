About Expense Tracking Web

Expense Tracking Web is a simple educational project developed for the CPIS 358 course. The main goal is to practice HTML, CSS and JavaScript by building a small website that helps users record their daily expenses.

The website includes forms for signup, login and adding expenses, tables to list the data, internal and external CSS for styling, and JavaScript functions for validation and simple calculations. It does not store real data in a database because it is a front-end practice project only.
